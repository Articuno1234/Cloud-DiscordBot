# Cloud-DiscordBot
![py](https://img.shields.io/badge/Python-3.5-blue.svg?style=social)

# Installing
```
git clone https://github.com/Articuno1234/Cloud-DiscordBot.git
```
then wait...
# Setting up
go into `config.py`
```
TOKEN = 'Your Bots Token'

PREFIX = "!"

OWNER = "Articuno#6618"
```
And put all the details in!
# Running
python3
```
cd Cloud-DiscordBot
python3 launcher.py
```
```
cd Cloud-DiscordBot
python3 cloud.py
```
python3.5
```
cd Cloud-DiscordBot
python3.5 Launcher.py
```
```
cd Cloud-DiscordBot
python3.5 cloud.py
```
